# taro-cnode

## 这是什么？

这是一个Taro 开发的多端客户端（h5， weapp，……）

你也可以把他看成是Taro开发指南， 

你也可把他看成React开发教程，

你还可以把他看成是Vue和React的比较开发教程。

因为UI和代码基本参照 https://github.com/shinygang/Vue-cnodejs，

你基本上可以找到跟 https://github.com/shinygang/Vue-cnodejs 一对一的目录结构。
